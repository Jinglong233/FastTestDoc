# 血缘追踪

血缘追踪用于可视化展示需求、用例、BUG 之间的关联关系。

## 追踪内容

从一个需求出发，可以展开看到：

- 直接关联的用例
- 用例发现的 BUG
- 直接关联的 BUG（不经过用例）

## 展示形式

采用 echarts 思维导图（TreeChart）展示，从左到右发散：

- 需求节点：蓝色
- 用例节点：绿色
- BUG 节点：红色

::: tip 截图占位
此处需补充「血缘追踪弹窗」截图，展示思维导图形式的需求-用例-BUG 关系。
:::

## 统计指标

弹窗顶部展示：

- 关联用例数
- 关联 BUG 总数
- 未关闭 BUG 数
- 用例通过率

## 交互

- 点击节点可折叠/展开
- 支持滚轮缩放画布
- 支持拖拽移动视图
- 鼠标悬浮显示完整名称和统计信息

## 筛选

支持「仅显示有 BUG 的用例」开关，快速聚焦问题分支。

## 相关接口

| 接口 | 说明 |
|------|------|
| `GET /qa/requirement/{id}/trace` | 获取需求血缘关系 |
