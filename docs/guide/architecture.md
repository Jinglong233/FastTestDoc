# 系统架构

FastTest 采用前后端分离架构。

## 技术栈

- **前端**：Vue 3 + Vite + TypeScript + Arco Design
- **后端**：Spring Boot + MyBatis-Plus + Sa-Token
- **数据库**：MySQL 8.0+
- **文档**：VitePress

## 模块划分

- **platform-starter**：启动入口
- **platform-automation**：基础平台 + UI/API 自动化 + 计划/报告/Mock/数据模板
- **platform-qa**：质量管理（需求、BUG、用例、测试计划）
