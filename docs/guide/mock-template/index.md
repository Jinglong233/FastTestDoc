# Mock 与数据模板

Mock 与数据模板是接口测试和自动化场景的两个重要辅助能力：

- **Mock**：在接口未就绪或不稳定时，模拟后端响应，让测试流程继续推进。
- **数据模板**：把常用的测试数据生成规则沉淀为可复用模板，快速构造批量数据。

## 使用场景

| 场景 | 推荐能力 | 说明 |
|------|---------|------|
| 后端接口还没开发完，想提前写用例 | Mock 响应 | 为接口配置固定 JSON 或字段规则生成的响应 |
| 依赖服务不稳定，自动化经常失败 | Mock 响应 | 启用 Mock 后，执行器直接返回模拟数据 |
| 需要构造大量用户/订单数据 | 数据模板 | 定义字段规则，批量生成 JSON/CSV/Excel |
| 接口参数需要随机手机号/UUID/枚举值 | 参数级 Mock | 在参数值上配置 Mock 规则，无需手写表达式 |

## 核心能力

### 接口级 Mock 响应

在 **API 测试** 页面编辑接口，切换到 **Mock** Tab：

- 启用 Mock 开关
- 设置响应状态码和延迟
- 配置响应 Header
- 用字段规则或原始 JSON 配置响应体

### 参数级 Mock

在接口请求的 Header / Query / Cookie / Body 等参数中：

- 参数值设为 `@phone()`、`@uuid()`、`@integer(0, 100)` 等 Mock 标识
- 执行时由后端按规则生成真实值并替换

### 数据模板

进入 **接口测试 → 数据模板**：

- 左侧为文件夹 + 模板混合树
- 右侧编辑模板字段规则
- 支持单条预览、批量生成、JSON/CSV/Excel 导出
- 可在请求体、Mock 响应、前后置脚本中引用
