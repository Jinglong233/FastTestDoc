# 团队管理

团队是 FastTest 的顶层组织单位。

## 创建团队

在团队列表页点击「新建团队」，填写团队名称、编号和描述即可创建。

::: tip 截图占位
此处需补充「团队列表页」和「新建团队弹窗」截图。
:::

## 切换团队

顶部导航栏的团队下拉框用于切换当前团队。切换团队时会自动清空当前项目上下文，避免请求中携带旧项目 ID。

::: tip 截图占位
此处需补充「顶部团队切换下拉框」截图。
:::

## 团队成员

每个团队有独立的成员列表，成员可以被分配不同角色。

- 支持邀请已有用户加入团队。
- 支持修改成员角色、状态、移除成员。
- 团队所有者或超管通常拥有团队管理权限。

::: tip 截图占位
此处需补充「团队成员管理页」截图。
:::

## 个人团队

每个用户默认拥有一个个人团队（`is_personal = true`），用于个人试用和私有项目。
